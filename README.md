## Neovim
### Requirements
- https://github.com/Shougo/dein.vim


```bash
 $ cd  ~
 $ git clone https://github.com/bernardocarvalho/dot-files
 $ mkdir -p ~/.config
 $ cd ~/.config
 $ ln -s ~/dot-files/nvim ./
 $ nvim
 :call dein#install()
```
